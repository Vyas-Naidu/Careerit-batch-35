id selector
*********
     id selector starts with "#"(hash)
     Syntax.
     ******

    HTML                       CSS 
    <p id="id1"></p>            #id1

what are The diff between class and id selector?

the class selector starts with"." and id starts with "#".
The diff between class and id selector is id selector have more priority(specificity) compared to class 
Class selector  may duplicate. but id selector must be unique.
USECASE:
        Class                                id
        in single project with multiple     to dominate 3rd party styles
                                             we wil use id selector
        teams are working then we choose 
        class selector.

Tables
*****
        collection of rows and columns 
    

    Tags
    ***
    1. <tables></table> used to create table.

    2. <tr></tr> used to create rows.
    3. <th></th> used to create table headings
                by default table headings in first row
    4. <td></td> used to create  a cell.

ATTRIBUTES
*********
1. border: used to draw broder to the table.
        Ex. 
        border="1"
2. align="left" / "center" / "right" 
        used to align the table
        "left is the default alignment"
        Ex.
            align="center"
 3. cell padding               
         space around the table cell content.
         Ex.
            cellpadding="10"
4. cellspacing 
            space around the table cell content.
            Ex.
                cellspacing="10"
5. colspan
            used to merge cells based on columns(horizontally)
            EX.
                colspan="2"
6. rowspan
           used to merge cells based on rows

Tags
***
<form action="" method="GET/POST/PUT/DELETE" name="" onsubmit="" onreset=""></form>
action attribute used to connect to backend
methods attribute supports four methods
    GET method, used to read data from backend
    POST method, used to send data to backend
    PUT method, used to update the data to backend
    DELETE method, used to DELETE backend data 

name attribute, used to assign logical name to the form
    Ex.
       name="regform"

onsubmit attribute,used to handle submit event

Ex. onsubmit ="register()"
     whenever we click the submit button,automatically the register() will execute in javascript

onresert attribute, used to handle reset event
Ex.
    onreset="cancel()"
      whenever we click the cancel button,automatically the cancel() will execute in javascript
